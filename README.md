A hubot script that does the things

See [`src/whois.coffee`](src/whois.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-whois --save`

Then add **hubot-whois** to your `external-scripts.json`:

```json
["hubot-whois"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
